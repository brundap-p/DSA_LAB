# Data Structures Laboratory Programs

This repository contains Data Structures laboratory programs completed as part of my academic coursework. The programs demonstrate the implementation of fundamental data structures and algorithms using programming concepts.

The laboratory exercises cover linear and non-linear data structures, along with basic searching and sorting techniques. These programs were developed to understand the design, implementation, and application of various data structures in problem-solving.

## Topics Covered

* Arrays
* Lists
* Stacks
* Queues
* Linked Lists
* Linked Stack
* Linked Queue
* Searching Techniques
* Sorting Techniques
* Binary Search Tree (BST)

## Purpose

The objective of this repository is to document and showcase practical implementations of core data structures and algorithms, providing a foundation for efficient data organization and processing.
